# Vulhub

This directory should contain the Vulhub repository.

## Installation

Clone Vulhub to this directory:

```bash
cd benchmark
rm -rf vulhub  # Remove this placeholder
git clone https://github.com/vulhub/vulhub.git
```

Or clone to home directory and create symlink:

```bash
git clone https://github.com/vulhub/vulhub.git ~/vulhub
cd benchmark
rm -rf vulhub
ln -s ~/vulhub vulhub
```

## About Vulhub

Vulhub is an open-source collection of pre-built vulnerable docker environments for security research and penetration testing.

- GitHub: https://github.com/vulhub/vulhub
- Documentation: https://vulhub.org/
